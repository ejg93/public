# 가운뎃점(·/ㆍ) — 공문서·법령 작성 표준 참고

> 특수문자_모음.html "가운뎃점·구분점" 카테고리 근거자료.
> 2026-07-15 정리. 폐쇄망 반입용 — 외부 링크 대신 본문에 요지 인용.

## 한 줄 결론

**표준이 하나가 아니다.** 문서 종류에 따라 쓰는 코드가 갈린다.

| 문서 종류 | 쓰는 글자 | 유니코드 | 근거 |
|-----------|-----------|----------|------|
| 일반 공문서·보고서·처리방침 문장 | 가운뎃점 `·` | **U+00B7** (MIDDLE DOT) | 국립국어원 「한글 맞춤법」 문장부호 |
| 법령·조례·규정·자치법규 조문 | 아래아 `ㆍ` | **U+318D** (한글 모음 아래아, HNC 357D) | 법제처 「알기 쉬운 법령정비기준」 |

겉보기 거의 같지만 유니코드가 다른 별개 문자다. 검색·치환·정렬에서 서로 안 잡힌다.

## 1. 국립국어원 — 문장부호 규정 (일반 문서용)

- 근거: 「한글 맞춤법」(문화체육관광부 고시 제2017-12호) 부록 '문장 부호'.
  원 개정 2014-10-27 고시, 2015-01-01 시행.
- 가운뎃점 **용법**만 규정하고 코드값은 지정하지 않음. 활자상 가운뎃점은 `·`(U+00B7).
- 주요 용법 세 가지:
  1. 짝을 이루는 어구 사이 — "등록·공개", "국내외", "시·도"
  2. 공통 성분을 줄여 하나로 묶을 때 — "통권 제54·55·56호"
  3. 열거할 어구를 묶을 때
- **대체 허용**: 위 경우 가운뎃점 대신 **쉼표(,)** 를 써도 된다. 특정한 의미의 날짜 표기(예: 3·1 운동)는 마침표(.)가 원칙, 가운뎃점 허용.
  → 컴퓨터 입력 편의 때문에 쉼표·마침표 허용으로 완화된 것.

정리: 일반 공문서·SR·처리방침 본문에서 "등록·공개"류는 **U+00B7** 쓰면 규정에 맞다. 쉼표로 바꿔도 무방.

## 2. 법제처 — 법령정비기준 (법령 조문용)

- 근거: 법제처 「알기 쉬운 법령 정비기준」(제10판). 상위 근거는 법제업무운영규정(대통령령).
- 법령에 가운뎃점을 넣을 때 **HNC(한컴2바이트) 코드 357D** = 전각 가운뎃점 = **아래아 ㆍ** 를 쓰도록 정함.
- 국가법령정보센터(law.go.kr)에 등재된 「대한민국헌법」 등 실제 법령 텍스트의 가운뎃점은 **U+318D** 로 저장돼 있음.
- 따라서 조례·규정·약관을 법령체로 정비할 때는 U+318D가 사내 표준일 수 있음. 발주처 지침 확인 필요.

## 3. 행정안전부 — 행정업무운영편람 (공문서 서식)

- 공문서 항목 기호 체계(1. → 가. → 1) → 가) → (1) → (가) → ① → ㉮)와 들여쓰기를 규정.
  → 이건 특수문자_모음.html "항목 기호 체계 > 공문서" 카드가 이미 반영.
- 문장부호 자체는 국립국어원 규정을 준용. 가운뎃점 코드 별도 명시는 약함.

## 4. 혼동하기 쉬운 유니코드 (겉모습 비슷, 전부 다른 글자)

| 글자 | 유니코드 | 이름 | 표준 용도 |
|------|----------|------|-----------|
| `·` | U+00B7 | MIDDLE DOT | 일반 문서 가운뎃점 (국어원) |
| `ㆍ` | U+318D | 한글 모음 아래아 | 법령 조문 가운뎃점 (법제처) |
| `ᆞ` | U+119E | 한글 중성 아래아 | 옛한글 조합용 — 단독 사용 금지 |
| `‧` | U+2027 | HYPHENATION POINT | 사전 음절 구분 — 공문서 부적합 |
| `・` | U+30FB | KATAKANA MIDDLE DOT | 일본어 전각 — 한국 공문서 부적합 |
| `∙` | U+2219 | BULLET OPERATOR | 수식 — 문장 가운뎃점 아님 |
| `•` | U+2022 | BULLET | 목록 불릿 — 문장 중간 부적합 |

## 5. 한컴오피스 한글(HWP) 입력 시 실제 들어가는 글자

- `Shift + ㅏ` → **아래아 ㆍ (U+318D)**  ← 법령용
- `Alt + 0183`(숫자 키패드) → **가운뎃점 · (U+00B7)**  ← 일반 문서용
- 두벌식 표준2 자판: 숫자 1 왼쪽 `` ` `` → · (U+00B7)
- 코드로 직접: 편집창에 `318D` 입력 후 `Alt+Shift+F10` → ㆍ 삽입 (문자표 Ctrl+F10에서도 16진 코드 입력 가능)

주의: 습관적으로 `Shift+ㅏ`로 치면 일반 문서에도 U+318D(아래아)가 들어간다. 폰트에 따라 위치·크기가 어긋나 보일 수 있으니, 일반 공문서는 U+00B7 권장.

## 6. 최근 흐름

- 2023 법무부 변호사시험 CBT: 아래아(U+318D) 배척, **U+00B7** 채택.
- 웹·모바일 환경에선 U+00B7이 점이 작고 자간이 좁아 가독성 논란 → 매체별 판단.
