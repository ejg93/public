intellij

| **분류** | **단축키** | **기능** |
| --- | --- | --- |
| **코드 작성** | **Ctrl** + **Space** | **코드 자동 완성** (Code Completion) |
|  | **Ctrl** + **Alt** + **Space** | 클래스, 메소드 이름 자동 완성 (Class/Method Name Completion) |
|  | **Ctrl** + **Shift** + **Enter** | **구문 완성** (괄호, 세미콜론, 중괄호 등 자동 추가) |
|  | **Alt** + **Enter** | **빠른 수정** (Quick Fix). 에러나 경고 발생 시 해결책 제시 |
| **검색 및 이동** | **Shift** 두 번 (Double Shift) | **모든 것 검색** (파일, 클래스, 설정, 액션 등 통합 검색) |
|  | **Ctrl** + **N** | **클래스 검색** |
|  | **Ctrl** + **Shift** + **N** | **파일 검색** (클래스가 아닌 일반 파일) |
|  | **Ctrl** + **B** 또는 **Ctrl** + **클릭** | **선언부(정의)로 이동** (Go to Declaration) |
|  | **Ctrl** + **Shift** + **F** | **프로젝트 전체에서 텍스트 검색** |
| **코드 정리** | **Ctrl** + **Alt** + **L** | **코드 포맷팅** (자동 정렬) |
|  | **Ctrl** + **/** | **한 줄 주석** 토글 (//) |
|  | **Ctrl** + **Shift** + **/** | **블록 주석** 토글 (/* ... */) |
| **코드 편집** | **Ctrl** + **D** | 현재 줄/선택 영역 **복제** (Duplicate) |
|  | **Ctrl** + **Y** | 현재 줄 **삭제** (Delete Line) |
|  | **Alt** + **Insert** | **생성** (Constructor, Getter/Setter, Override Method 등) |
|  | Shift + Alt + U | **카멜 표기법 적용** |
| **리팩토링** | **Ctrl** + **Alt** + **V** | **변수 추출** (Extract Variable) |
|  | **Ctrl** + **Alt** + **M** | **메소드 추출** (Extract Method) |
| **화면 관리** | **Alt** + **1** | **프로젝트 창 토글** (Project Window Toggle) |

크롬 디버그

F8 (Resume) - 추가 중단점까지 실행을 재개한다.

F9 (Step) - 스크립트 전체를 문 단위로 하나하나 실행할 수 있다.

F10 (Step Over) - 다음 명령어를 실행하되, 함수 안으로 들어가진 않음.

F11 (Step Into) - 다음 명령어를 실행하되 비동기 동작이 완료될 때까지 대기한다.

Shift+F11 (Step Out) - 실행중인 함수의 실행이 끝날 때까지 실행을 계속함.

현재 실행중인 함수의 실행을 계속 이어나가다가 마지막 줄에서 실행을 멈춘다.
