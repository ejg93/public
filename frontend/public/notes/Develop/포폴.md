1-1. 기술 스택 확정

- 프론트엔드: Next.js + Tailwind
- 백엔드: Java Spring Boot 3.x
- 데이터베이스: MySQL
- 인프라: Docker, Nginx 1-2. 외부 API 리스트업
- OpenAI/Gemini
